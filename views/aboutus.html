  1. Programming Timeline

    Today is {{date | date:"MMMM dd'th,' yyyy"}}

    This is a timeline of my programming career. It ranges from June 2015 to June 2017. You can navigate by using the arrow keys or click on each date and circle individually.

  2. Discovery of MOOC Courses

    June 2015

    MOOC stands for Massive Online Open Courses. In the summer '15 I came across a post on r/learnprogramming titled "100+ programming courses you can start today for FREE". Up to that point it was my assumption was that the only way one could learn programming was to go to an accredited university and get a four year computer science degree. I was overwhelmed by the resources these MOOC courses provided, and this over exposure to material was something I struggled with for a while. In took me a about year of trying out different online courses to find a direction I was comfortable with.

  3. CS50x

    September 2015

    CS50x is a very well made online learning resource. It teaches the fundamentals of computer science and also transitions into web development. I messed around with this course by watching all the lectures and doing some of the homework assignments particularly regarding web development.

  4. Joined GitHub

    September 9th, 2015

    Day I officially made my GitHub account. Not sure what was the reason at that time.

  5. Python Projects

    December 2015

    During this time I completed the Learn Python the Hard Way tutorial and also attempted the Tango with Django. The reason for learning Python and Django was my brother, at the time he was using the language and framework for his work and expressed favorably for the stack. I ended up working with him later on so knowledge of Python and Django came in handy.

  6. Initial Github Commit

    February 18th 2016

    Around this time I began the first course of the Coursera Full Stack Specialization. To this day this is my favorite learning source because it provided me adequate means of direction. Coursera Full Stack Specialization allowed me to finally trust a resource that I felt comfortable committing time to.

  7. Coursera Bootstrap Course

    March 2016

    Began working on Coursera's full stack specialization front end courses one and two starting with basic html css and transitioning to Bootstrap. Around this time I started building initial layouts for the conFusion app website that I further develop enhancements to throughout the latter Coursera courses.

  8. Depaul Web Development Program

    April 2016

    Started Depaul's Web Development with HTML5 and JavaScript Program. A 3 month program that covered web developing with javascript and the new features of HTML5 and CSS3. Here is where I built my first website Chicago. The program also strengthened my fundamentals of Javascript programming.

  9. Python Internship

    June 2016

    An internship I did with my brother where I helped him manage data files and built UI widgets for his Bitcoin trading software. Technologies used were Python and Django in addition with Ember.

  10. WordPress Internship

    September 2016

    I worked with WordPress for a small design team called what2design. In this internship I interacted with clients and built custom WordPress themes.

  11. Coursera AngularJS Course

    January 2017

    Started working through the Coursera Full Stack Specialization AngularJS Course #3. After this course I became to understand the roll of a Front-End developer in much greater detail.

  12. Portfolio Website

    March 2017

    After figuring out how to host websites on GitHub Pages, I started hosting my online projects and began working on this porfolio website. This website is a Single Page Application made with AngularJS. It is made responsive with Bootstrap and jQuery. This portfolio website was made to serve as a midpoint project in the coursera full stack web specialization program.

  13. First Pull Request

    March 2017

    Contributed to my first open source project by building a simple Budget UI that implements RESTful API to pull a budgets object from a remote server and have it displayed in browser. The app allows basic visual manipulation of said object by adding updating and removing fields both on the front end and simultaneously on the back end.

  14. Lambda Javascript Mini Bootcamp

    July 2017

    Finished a javascript bootcamp mini series to sharpen understanding of javascript object oriented programming, syntax, caveats, key characteristics, and tricks of the trade.

  15. GCS Fund

    August 2017

    Started working for a cryptocurrency trading company where I developed custom javascript buttons and inputs for an online trading platform.
