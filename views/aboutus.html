  1. Programming Timeline

    Today is {{date | date:"MMMM dd'th,' yyyy"}}

    This is a timeline of my programming career. It ranges from June 2015 to June 2017. You can navigate using the arrow keys or click on each date or circle individually.

  2. Discovery of MOOC Courses

    June 2015

    MOOC stands for Massive Online Open Courses. In the summer '15 I came across a post on r/learnprogramming titled "100+ programming courses you can start today for FREE". Up to that point it was my assumption was that the only way one could learn programming was to go to an accredited university and get a four year computer science degree. I was overwhelmed by the resources these MOOC courses provided, and this over exposure to material was something I struggled with for a while. In took me a about year of trying out different online courses to find a direction I was comfortable with.

  3. CS50x

    September 2015

    CS50x is a very well made online learning resource. It teaches the fundamentals of computer science very well and transitions into web development. I messed around with this course by watching all the lectures and doing some of the homework assignments regarding web development that is introduced later in the course.

  4. Joined GitHub

    September 9th, 2015

    Day I officially made my GitHub account. I'm not sure what was the reason at that time.

  5. Python Projects

    December 2015

    During this time I completed the Learn Python the Hard Way tutorial and also attempted the Tango with Django tutorial. The reason for learning Python and Django was my brother at the time was using the language and framework for his work and expressed favorably for the stack. I ended up working with him on the same stack later on so knowledge of Python and Django came in handy.

  6. Github Initial Commit

    February 18th 2016

    Initial commit on GitHub. Around this time I began the first course of the Coursera Full Stack Specialization. This learning source is to this day my favorite source because it provided adequate means of direciton for me. Coursera Full Stack Specialization allowed me to finally trust a resource that I felt comfortable committing my time to.

  7. Coursera Bootstrap Course

    March 2016

    Began working on coursera's full stack specialization front end courses starting with bootstrap and transitioning to AngularJS. Around this time I started building intitial layouts of the conFusion app website that I further develop enhancements on throughout the latter Coursera courses.

  8. Depaul Web Development Program

    April 2016

    Was accepted to Depaul's Web Development with HTML5 and JavaScript Program. This was a 3 month program that covered web developing with javascript and the new features of HTML5 and CSS3. Here is where I built my first website Chicago. This program also strengthened my fundamentals of Javascript programming.

  9. Python Internship

    June 2016

    An internship I did with my brother were I helped him manage data files and built UI widgets for a Bitcoin trading software. Technologies used were Python and Django in addition with Ember.

  10. WordPress Internship

    September 2016

    I worked with WordPress for a small design team called what2design. In this internship I interacted with clients and built custom WordPress themes.

  11. Coursera AngularJS Course

    January 2017

    Started working through the Coursera Full Stack Specialization AngularJS Course #3. After this course I became to understand the roll of a Front-End developer in much greater detail.

  12. Portfolio Website

    March 2017

    After figuring out how to host websites on GitHub Pages, I started hosting my online projects and began working on this porfolio website. This website is a Single Page Application made with AngularJS. It is made responsive with Bootstrap and jQuery. This portfolio website was made to serve as a midpoint project in the coursera full stack web specialization program.

  13. First Pull Request

    March 2017

    Contributed to my first open source project by building a simple Budget UI that implements RESTful API to pull a budgets object from a remote server and have it displayed in browser. The app allows basic visual manipulation of said object by adding updating and removing fields both on the front end and simultaneously on the back end.
